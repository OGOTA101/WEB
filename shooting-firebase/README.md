# 4方向シューティングゲーム - SF Mission

Firebase Realtime Databaseを使用したリアルタイムマルチプレイヤーシューティングゲームです。

## 🎮 ゲーム概要
- **マルチプレイヤー**: ブラウザを変えてアクセスすると、みんなで協力して戦えます。
- **4方向ショット**: 停止しているときだけ、上下左右に攻撃できます。
- **タップ移動**: 画面をタップ/クリックした場所に移動します。
- **SFテーマ**: 宇宙空間でエイリアンの大群から生き残れ！

## 🕹️ 操作方法
- **移動**: 画面をクリック/タップ
- **攻撃**: 自動（停止時のみ）

## 🛠️ 技術スタック
- HTML5 Canvas
- JavaScript (Vanilla)
- Firebase Realtime Database

## 🚀 実行方法
```bash
firebase serve
```
