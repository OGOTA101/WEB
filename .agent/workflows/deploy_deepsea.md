// turbo-all

Deploys the Deep Sea Survival game to Firebase Hosting.
**Note:** This project shares the `sggame-hub` ID with the main HP. Deploying this may overwrite the HP content on the default site.

## Steps

1. Navigate to directory and deploy
```bash
cd c:\Users\ihogo\Desktop\WEB\deepsea-firebase
firebase deploy --only hosting
```

2. Verification
- https://sggame-hub.web.app/
